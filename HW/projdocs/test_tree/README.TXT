This test directory tree contains:

bash-4.2/
  Source code for bash version 4.2, available online at:
  http://tiswww.case.edu/php/chet/bash/bashtop.html

books/
  Text for various public domain books, downloaded from
  Project Gutenberg (http://www.gutenberg.org/).

enron_email/
  A small subset of the public domain Enron email data
  set, downloaded from http://www.cs.cmu.edu/~enron/

jan6-report/
  The text of the Select January 6th Committee Final Report, downloaded from
  https://www.govinfo.gov/collection/january-6th-committee-final-report and
  converted from .pdf into raw text.

tiny/
  A directory consisting of very small input files.  This directory's
  contents are small enough that you should be able to draw, on paper, the
  full contents of all HW2 data structures if searchshell were to crawl
  this directory.

example.txt
  The example text file from the HW2 web page.
